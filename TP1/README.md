### Instrucciones para correr el proyecto:

- Para compilar:
```bash
gcc src/pokedex.c tp1.c -o tp1
```

- Para ejecutar:
```bash
./tp1 <nombre_archivo>
```

- Para ejecutar con Valgrind:
```bash
valgrind ./tp1 <nombre_archivo>
```
